Tooltip and Popover - Grupo 01

Teoría

Tooltip y Popover: Componentes interactivos para mejorar la experiencia de usuario.

¿Qué es un Tooltip?

Un tooltip es un pequeño elemento emergente que aparece cuando el usuario coloca el cursor sobre un elemento específico. Proporciona información adicional sin ocupar espacio permanente en la interfaz.

¿Qué es un Popover?

Un popover es un contenedor más grande que aparece al hacer clic o interactuar con un elemento. A diferencia del tooltip, puede contener texto formateado, botones, imágenes y otros elementos HTML.

Diferencias principales:
  • Activación: Tooltips se activan al pasar el cursor (hover), popovers requieren clic.
  • Contenido: Tooltips contienen texto simple, popovers pueden incluir HTML complejo.
  • Duración: Tooltips desaparecen al quitar el cursor, popovers permanecen hasta ser cerrados.
  • Tamaño: Tooltips son pequeños y discretos, popovers pueden ser más grandes.
Casos de uso:

Tooltips: Explicar iconos, mostrar información adicional sobre campos de formulario, proporcionar ayuda contextual.

Popovers: Formularios pequeños, menús de configuración, información detallada, confirmaciones de acciones.

Ejemplos

Ejemplo práctico de Tooltips con Bootstrap

Demostración interactiva de tooltips y popovers utilizando Bootstrap 5.

En este fragmento se está mostrando cómo funcionan los Tooltips en Bootstrap, que básicamente son pequeños cuadros de texto que aparecen cuando el usuario pasa el cursor sobre un elemento, en este caso un botón. Cada botón tiene el atributo data-bs-toggle="tooltip", que le indica a Bootstrap que debe activar esta función, y el atributo title, que contiene el texto que se mostrará dentro del tooltip. Además, con data-bs-placement se define la posición en la que aparecerá el mensaje —puede ser arriba (top), abajo (bottom), a la izquierda (left) o a la derecha (right)—. Por ejemplo, el botón con data-bs-placement="left" mostrará el texto “Izquierda” al lado izquierdo del botón. Es importante mencionar que para que los tooltips funcionen correctamente no basta solo con el HTML, también se necesita tener cargado el JavaScript de Bootstrap y una pequeña inicialización que los active en la página. En conjunto, este código sirve como una demostración práctica de cómo usar los tooltips para ofrecer información adicional de forma visual y elegante.

Código HTML:
<div class="container">
                        <h1>Tooltips</h1><br>
                        
                        <div class="row">
                            <div class="col-xs-12 col-sm-3">
                                <button class="btn btn-primary btn-block" data-bs-toggle="tooltip" 
                                        data-bs-placement="left" title="Izquierda">Presionar 1</button>
                            </div>
                            <div class="col-xs-12 col-sm-3">
                                <button class="btn btn-primary btn-block" data-bs-toggle="tooltip" 
                                        data-bs-placement="top" title="Superior">Presionar 2</button>
                            </div>
                            <div class="col-xs-12 col-sm-3">
                                <button class="btn btn-primary btn-block" data-bs-toggle="tooltip" 
                                        data-bs-placement="right" title="Derecha">Presionar 3</button>
                            </div>
                            <div class="col-xs-12 col-sm-3">
                                <button class="btn btn-primary btn-block" data-bs-toggle="tooltip" 
                                        data-bs-placement="bottom" title="Inferior">Presionar 4</button>
                            </div>
                        </div>
                    </div>

Código HTML para Popovers:

Ejemplo práctico de Popovers con Bootstrap

En este código se está utilizando la funcionalidad de Popovers de Bootstrap, que son similares a los tooltips pero más avanzados, ya que pueden mostrar no solo una pequeña etiqueta de texto sino también contenido más completo, como párrafos, enlaces o cualquier otro texto adicional. Cada botón tiene el atributo data-bs-toggle="popover", que le indica a Bootstrap que debe generar un popover al interactuar con él, y data-bs-content, donde se define el texto o contenido que se mostrará dentro del recuadro emergente. Además, el atributo data-bs-placement controla en qué posición aparece el popover (izquierda, arriba, derecha o abajo del botón), mientras que data-container="body" especifica que el popover se renderice dentro del cuerpo del documento, evitando problemas de posicionamiento con otros elementos del DOM. En conjunto, este fragmento crea cuatro botones con estilo btn btn-info, cada uno mostrando un popover en una dirección diferente cuando el usuario hace clic. Al igual que con los tooltips, es necesario tener el JavaScript de Bootstrap cargado e inicializar los popovers para que se activen correctamente en la página.

<div class="container">
                            <h1>Popover</h1><br>
                            
                            <div class="row">
                                <div class="col-xs-12 col-sm-3">
                                    <button class="btn btn-info btn-block" data-container="body" 
                                            data-bs-toggle="popover" data-bs-placement="left" data-bs-content="Lorem">Presionar 1</button>
                                </div>
                                <div class="col-xs-12 col-sm-3">
                                    <button class="btn btn-info btn-block" data-container="body" 
                                            data-bs-toggle="popover" data-bs-placement="top" data-bs-content="Lorem">Presionar 2</button>
                                </div>
                                <div class="col-xs-12 col-sm-3">
                                    <button class="btn btn-info btn-block" data-container="body" 
                                            data-bs-toggle="popover" data-bs-placement="right" data-bs-content="Lorem">Presionar 3</button>
                                </div>
                                <div class="col-xs-12 col-sm-3">
                                    <button class="btn btn-info btn-block" data-container="body" 
                                            data-bs-toggle="popover" data-bs-placement="bottom" data-bs-content="Lorem">Presionar 4</button>
                                </div>
                            </div>
                        </div>
