Progress Bar - Grupo 05

Las barras de progreso son elementos visuales diseñados para mostrar el avance de una tarea de manera clara y dinámica. Su objetivo es ofrecer al usuario una sensación de control y fluidez mientras espera la carga, descarga o ejecución de un proceso. Más allá de su función técnica, también aportan estética y coherencia visual al diseño del sitio web. Gracias a tecnologías como HTML, CSS y JavaScript o frameworks como Bootstrap, es posible crear barras lineales, circulares o animadas que no solo informan, sino que también mejoran la experiencia del usuario al transformar el tiempo de espera en una interacción más agradable.

Video Tutorial

Las barras de progreso se crean con elementos HTML y CSS, utilizando contenedores personalizados que permiten apilar, animar y añadir texto. El elemento .progress actúa como contenedor general, mientras que .progress-bar muestra el avance mediante su ancho ajustable.

En esta pagina hay varios ejemplos con codigo y ejecucion que ilustran los distintos estilos de las barras de progreso.

Ejemplo 1: Código del Video - Múltiples Barras

Este ejemplo muestra una barra de progreso compuesta, donde se utilizan varias sub-barras con diferentes colores para representar distintos estados o etapas de un proceso. Cada una usa una clase de color de Bootstrap (bg-success, bg-warning, bg-danger) para indicar niveles de éxito, advertencia o error dentro del mismo progreso. 

<div class="progress">
    <div class="progress-bar bg-success" style="width:35%">
        <span class="sr-only">35% completado (éxito)</span>
    </div>
    <div class="progress-bar bg-warning" style="width:20%">
        <span class="sr-only">20% completado (aviso)</span>
    </div>
    <div class="progress-bar bg-danger" style="width:10%">
        <span class="sr-only">10% completado (peligro)</span>
    </div>
</div>
Ejemplo 2: Barra de Progreso con Etiqueta

Esta barra de progreso incluye un porcentaje visible dentro de la barra. Muestra el valor actual de progreso directamente en el elemento, útil para indicar visualmente el avance de una tarea específica. 

<div class="progress">
    <div class="progress-bar" role="progressbar" style="width: 30%;" aria-valuenow="30" 
        aria-valuemin="0" aria-valuemax="100">30%
    </div>
</div>
Ejemplo 3: Barra de Progreso Animada

Este ejemplo utiliza una barra animada con franjas diagonales, lograda mediante las clases progress-bar-striped y progress-bar-animated. Simula movimiento continuo, ideal para indicar procesos en ejecución. 

<div class="progress">
    <div class="progress-bar progress-bar-striped progress-bar-animated" role="progressbar"
         aria-valuenow="75" aria-valuemin="0" aria-valuemax="100" style="width: 75%">
    </div>
</div>
Ejemplo 4: Barras de Progreso con Diferente Altura

En este caso se modifican las alturas de las barras con el atributo style="height:". Esto permite crear barras más delgadas o más gruesas dependiendo del contexto visual deseado. 

<div class="progress mb-2" style="height: 1px;">
    <div class="progress-bar" role="progressbar" style="width: 25%;" 
         aria-valuenow="25" aria-valuemin="0" aria-valuemax="100"></div>
</div>

<div class="progress" style="height: 20px;">
    <div class="progress-bar bg-success" role="progressbar" style="width: 50%;" 
         aria-valuenow="50" aria-valuemin="0" aria-valuemax="100"></div>
</div>

Este ejemplo muestra cómo usar las clases de color de Bootstrap como bg-success, bg-info, bg-warning y bg-danger para representar diferentes estados o niveles de avance. Cada barra está separada con la clase mb-2.

<!-- Script Barra de Progreso Formulario-->
<script>
      document.addEventListener('DOMContentLoaded', function () {
        const btnFormulario = document.getElementById('btnFormulario');
        const modalCargando = new bootstrap.Modal(document.getElementById('modalCargando'));
        const modalFormulario = new bootstrap.Modal(document.getElementById('modalFormulario'));
        const progressBar = document.getElementById('progressBarCarga');

        btnFormulario.addEventListener('click', (e) => {
          e.preventDefault();

          let progreso = 0;
          progressBar.style.width = '0%';
          modalCargando.show();

          const intervalo = setInterval(() => {
            progreso += 10;
            progressBar.style.width = progreso + '%';
            progressBar.setAttribute('aria-valuenow', progreso);

            if (progreso >= 100) {
              clearInterval(intervalo);
              setTimeout(() => {
                modalCargando.hide();
                modalFormulario.show();
              }, 400);
            }
          }, 200);
        });
      });
    </script>

Al realizar clic en el botón Formulario, se ejecuta un proceso visual que despliega un modal de carga. En donde se encuentra una barra de progreso definida mediante la clase .progress-bar y estilizada con el color verde (.bg-success). A través de un bloque de JavaScript, se controla la animación de llenado de la barra mediante un incremento progresivo del atributo width, simulando el proceso de carga de un formulario. Una vez que la barra alcanza el 100%, el script detiene el intervalo de actualización, oculta el modal de carga y muestra un segundo modal que confirma que el formulario de registro ha sido cargado correctamente.

      let progreso = 0;
      progressBar.style.width = '0%';
      progressBar.textContent = '0%';
      modalCargaTabla.show();

      const intervalo = setInterval(() => {
        progreso += 10;

        // Colores dinámicos según el progreso
        if (progreso < 50) {
          progressBar.className = 'progress-bar bg-danger progress-bar-animated';
        } else if (progreso < 80) {
          progressBar.className = 'progress-bar bg-warning progress-bar-animated';
        } else {
          progressBar.className = 'progress-bar bg-success progress-bar-animated';
        }

        progressBar.style.width = progreso + '%';
        progressBar.setAttribute('aria-valuenow', progreso);
        progressBar.textContent = progreso + '%';

        if (progreso >= 100) {
          clearInterval(intervalo);
          setTimeout(() => {
            modalCargaTabla.hide();
            modalTabla.show();
          }, 400);
        }
      }, 150);
    });
  });
</script>
