List - Grupo 06

Teoría

Listas

Una lista en HTML es una estructura utilizada para agrupar ítems relacionados o no relacionados. Su propósito principal es organizar la información de una manera clara y estructurada.

  • Estructura: Se define mediante un contenedor principal (ul para desordenada, ol para ordenada o dl para descripción) que agrupa una serie de ítems secundarios (li).
  • Concepto: Representa una colección de elementos donde cada elemento (li) tiene una relación o dependencia con el conjunto. Es la base para crear menús, navegaciones y contenidos enumerados.

Tipos de Listas

Listas No Ordenadas

Son las más sencillas y las más utilizadas. Es un conjunto de elementos relacionados entre sí, pero que no tienen un orden o secuencia determinados.

ul: encierra todos los elementos de la lista.
li: encierra cada uno de los elementos. 

<ul>
  <li>Café</li>
  <li>Té</li>
</ul>
Listas Ordenadas

Son similares a las listas no ordenadas, pero los elementos se muestran siguiendo un orden determinado.

ol: lista ordenada.
li: encierra cada uno de los elementos. 

<ol>
  <li>Paso 1</li>
  <li>Paso 2</li>
</ol>
Listas de Elementos Complejos

Son componentes flexibles que permiten mostrar elementos conformados por distintos tipos de contenido HTML. Se puede incluir prácticamente cualquier tipo de elemento dentro de cada ítem.

Class: list-group-item-heading
Class: list-group-item-text 

<div class="list-group">
    <a href="#" class="list-group-item active">
        <h4 class="list-group-item-heading">Mercedes-Benz A-Class</h4>
        <p class="list-group-item-text">
            La A-Class representa la entrada de gama de Mercedes-Benz.
        </p>
    </a>
    <a href="#" class="list-group-item">
        <h4 class="list-group-item-heading">Mercedes-Benz C-Class</h4>
        <p class="list-group-item-text">
            La C-Class es un sedán compacto-mediano.
        </p>
    </a>
</div>
Listas de Elementos con Badge

Muestran una cantidad o un estado asociado al ítem principal de la lista.

Class: list-group
Class: list-group-item 

<li class="list-group-item d-flex...">
  Imágenes
  <span class="badge">10</span>
</li>
Listas de Elementos de Enlace

Para crear una lista de elementos interactivos se reemplazan los elementos li por elementos a. Se encierran en un contenedor div (no en ul). En este tipo de listas se visualiza un efecto hover en cada elemento. Si se desea una lista activa, se utiliza la clase active en el elemento correspondiente.

Class: list-group
Class: list-group-item 

<div class="list-group">
  <a href="#" class="list-group-item">
    Inicio
  </a>
</div>
Operaciones jQuery por Tipo de Lista

jQuery es una biblioteca de JavaScript que facilita la manipulación de elementos HTML (el DOM). Para las listas, nos permite agregar, eliminar y modificar ítems (<li>) dentro de listas ordenadas (<ol>) o desordenadas (<ul>) de forma dinámica.

Selectores Clave

Los selectores son la base de jQuery, ya que le indican qué elementos del DOM (Document Object Model) debe manipular. jQuery utiliza la sintaxis de selectores CSS para encontrar elementos.

  • $("ul"): Selecciona todas las listas desordenadas.
  • $("#mi-lista"): Selecciona el elemento con el ID "mi-lista".
  • $(".clase-item"): Selecciona todos los elementos con la clase "clase-item".
  • $("#mi-lista li"): Selecciona todos los <li> dentro de la lista "#mi-lista".

Métodos Fundamentales

Los métodos fundamentales son el corazón de la interactividad en jQuery. Son los que ejecutan las acciones sobre los elementos que previamente seleccionaste.

  • .append("..."): Añade contenido (ej. un nuevo <li>) al final del elemento seleccionado.
  • .prepend("..."): Añade contenido al principio del elemento seleccionado.
  • .remove(): Elimina el elemento seleccionado (y todo su contenido).
  • .on("click", function() { ... }): Asigna un evento de clic. Es fundamental para la interactividad.
  • .toggleClass("clase"): Añade una clase CSS si no la tiene, o la quita si ya la tiene. Perfecto para "marcar" ítems.
  • .val(): Obtiene o establece el valor de un campo de formulario (como un <input>).
  • .parent(): Selecciona el elemento "padre" directo del elemento actual.

Evento Document Ready

En jQuery se utiliza para ejecutar código cuando el documento HTML ha sido completamente cargado y parseado. Este evento se dispara cuando todos los elementos del DOM están disponibles para manipular.

$(document).ready(function() {
// Funcionalidad
});

Esta función se encarga de aumentar el valor numérico del badge asociado al elemento seleccionado.


Código Acción
const $itemResaltado = $badgeList.find(".resaltado"); Busca el ítem seleccionado (el li o a con la clase .resaltado).
const $badge = $itemResaltado.find(".badge"); Una vez encontrado el ítem resaltado, busca el badge (span con clase .badge) dentro de él.
let valorActual = $badge.text(); Contenido actual del badge.
let nuevoValor = parseInt(valorActual, 10) + 1; Convierte el texto a un número entero y le suma 1.
$badge.text(nuevoValor); Actualiza el texto del badge con el nuevo valor incrementado.
